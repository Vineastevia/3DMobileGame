# Jeu Mobile 3D en Godot
Jeu développé sous Godot réalisé dans le cadre d'étude en master 1 info à l'USMB (INFO701).

## Description

Il s'agit d'un jeu sérieux 3D pour mobile avec pour thème un univers de Donjons et l'initiation de notre joueur au sein de l'Ordre des Mages-Codeur. Il découvrira les notions de programmations Python en résolvant des enigmes pour avancer dans la tour.

## Getting Started
